AI Scheduler 🤖

Overview 📌

This is a PoC project demonstrating how LLM APIs like ChatGPT or DeepSeek can be integrated into an existing system, for example, a small beauty boutique 💅 that provides services:

    Client Interaction 🗣️

    • 📅 Schedule a new appointment.
    • 🔄 Reschedule an existing appointment.
    • ⏳ Inform about a delay.
    • ❌ Cancel an appointment.

    Simulation Mechanics ⏳

    This is a time-accelerated simulation. You can freeze the clock or set a target time ⏰ for testing. In this system:

    • ⏱️ 5 real-world seconds equal 1 simulation minute.
    • ⏳ Time automatically resets to 9 AM the next day after reaching 4:05 PM.
    • 📆 Friday transitions directly to Monday.
